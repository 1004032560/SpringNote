1. 检测内核版本，必须是 3.10 及以上

   `[root@hadoop01 ~]# uname -r`

2. 安装 Docker

   `[root@hadoop01 ~]# yum install docker`

3. 输入 `y` 确认安装

4. 启动 Docker

   `[root@hadoop01 ~]# systemctl start docker`

5. 开机启动 Docker

   `[root@hadoop01 ~]# systemctl enable docker`

6. 停止 Docker

   `[root@hadoop01 ~]# uname -r`
