## IO流



Java IO 的学习是一件非常艰巨的任务。

构建这个体系又需要深入理解IO库的演进过程，所以，我们如果缺乏历史的眼光，很快我们会对什么时候应该使用IO中的哪些类，以及什么时候不该使用它们而困惑。



IO 库就是在前人的基础上不断优化总结提炼的一个结果。为了更好地学习或者使用这个库，需要去追溯他的历史和他的演变过程，程序都是在不断的优化兼容中发展的。
